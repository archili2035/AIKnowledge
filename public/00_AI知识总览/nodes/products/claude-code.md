## 是什么
Claude Code 可以先理解为 Anthropic 面向编程场景提供的一种代码助手 / Code Agent 形态，重点在于把模型能力落到读代码、改代码和执行任务的工作流里。

## 归类
它属于“产品层 / 代码助手层”，而不是单纯的模型节点。和 Claude 这类模型名称相比，它更接近开发者真正操作的工具形态。

## 为什么重要
代码助手是 Agent 化最明显的一条产品线。把 Claude Code 单独成条，能帮助知识图谱解释：为什么“会说”正在逐渐变成“能读、能改、能执行”。

## 观察与判断
这类产品的关键差异，不只在模型本身，还在上下文理解、工具接入和执行闭环。后续如果继续补条目，建议把“底座模型能力”和“工具层体验”继续分开记录。
