## 是什么
Cursor 可以先理解为一种面向编程场景的 AI IDE / 代码助手产品形态，重点在于把代码理解、生成、修改与工作流操作整合进同一界面。

## 归类
它属于“代码助手 / Code Agent”这条产品线，不是底层模型，也不是抽象概念本身。

## 为什么重要
代码助手是很多人最早感受到“AI 不只是会说，还会动手”的入口之一，因此在总览里很适合作为独立节点。

## 观察与判断
这类产品和通用聊天助手相比，更强调上下文工程、编辑器协同、代码库理解与任务闭环，因此也天然更容易向 Agent 化方向继续生长。
