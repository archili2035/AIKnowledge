# HTML视觉统一关键词（AIKnowledge）

> 适用范围：`d:/AIKnowledge` 下公开与私有页面（除内容特殊页可局部例外）

## 1) 设计关键词

- 渐变科技感（蓝紫主色）
- 卡片化信息块
- 高留白 / 高可读
- 柔和阴影层级
- 状态标签化（公开/私有/来源）
- 清晰入口（少跳转、少歧义）

## 2) 基础视觉令牌（Tokens）

```css
:root {
  --primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --accent: #4361ee;
  --bg: #f5f7fb;
  --card: #ffffff;
  --text: #1a1a2e;
  --subtext: #5a6a7a;
  --border: #e8ecf1;
  --shadow-sm: 0 2px 10px rgba(16, 24, 40, 0.04);
  --shadow-md: 0 8px 24px rgba(16, 24, 40, 0.08);
}
```

## 3) 页面结构建议

1. Hero（标题 + 一句话定位）
2. 核心入口区（卡片网格）
3. 内容主体（分节卡片）
4. 补充信息（来源、分级、更新时间）

## 4) 组件约束

- 卡片圆角：`12px ~ 18px`
- 卡片边框：`1px solid var(--border)`
- 主标题字号：`28px ~ 32px`
- 正文字号：`14px ~ 16px`
- 行高：`1.7 ~ 1.9`
- 链接样式：主色 + hover 下划线

## 5) 文案与信息架构规范

- 公共页只保留公开信息，不出现私有敏感正文。
- 私有页必须显示“私有级”状态标识。
- 所有页面优先提供“返回上级入口”链接，减少迷路。
- 避免过度装饰：动画与渐变仅用于层级提示，不用于干扰阅读。
